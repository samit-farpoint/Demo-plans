Drag-and-Drop User Flow
1
🖱️ User Grabs Drag Handle
Cursor changes to "grab" on hover over drag handle (≡ icon)
Visual: Handle highlights, cursor shows grab icon
2
📤 Drag Initiated
DndContext creates drag preview (ghost item), original item becomes semi-transparent
Visual: Ghost follows cursor, drop targets highlight
3
🎯 Hover Over Target Position
SortableContext calculates new position, shows drop indicator line between items
Visual: Blue line appears between target items
4
🖱️ User Releases (Drop)
onDragEnd fires, calculates new order array, calls reorderTodos API
Visual: Item snaps to new position with animation
5
✅ Order Persisted
Backend updates database, UI optimistically shows new order immediately
Visual: Smooth transition, all items settle into new positions
🔄 Error Recovery Flow
If API call fails:
1. Show error toast notification
2. Revert UI to previous order (optimistic rollback)
3. Allow user to retry drag operation
